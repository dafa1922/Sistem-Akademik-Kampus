# 🎓 Sistem Informasi Akademik Kampus (SIAKAD)

Sistem Informasi Akademik berbasis Desktop yang dibangun menggunakan **Java Swing** dengan tema modern **FlatLaf**. Aplikasi ini dirancang untuk mengelola data mahasiswa, dosen, mata kuliah, hingga proses transaksi KRS dan penilaian secara efisien.

## ✨ Fitur Utama
* **Dashboard Informatif:** Statistik real-time jumlah data dan jam digital.
* **Manajemen Master Data:** Kelola data Mahasiswa, Dosen, dan Mata Kuliah.
* **Sistem Transaksi:** Input KRS (Kartu Rencana Studi) dan Input Nilai Mahasiswa.
* **Hak Akses (Session):** Login dengan level Admin dan Operator.
* **Laporan PDF:** Cetak struk KRS otomatis menggunakan library iText.
* **UI Modern:** Menggunakan FlatLaf Mac Dark Theme agar nyaman di mata.

---

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Java (JDK 23)
* **IDE:** NetBeans Apache
* **Database:** MySQL (XAMPP)
* **Library Eksternal:**
  - `FlatLaf` & `FlatLaf-IntelliJ-Themes` (Untuk UI)
  - `MySQL Connector Java` (Koneksi Database)
  - `iText PDF` (Untuk cetak laporan)
